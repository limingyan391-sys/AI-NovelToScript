# ✦ NovelToScript

> **AI 辅助剧本创作工具** — 将小说文本自动转换为结构化剧本（YAML 格式），降低改编门槛，提升创作效率。

## 📖 简介

每一位小说作者都希望自己的作品能搬上银幕，但将小说改编为剧本是一项专业且耗时的工作。**NovelToScript** 应运而生：

- **自动解析** 小说章节、对话、场景信息
- **智能转换** 为符合行业规范的剧本格式
- **可视化编辑** 角色管理、场景时间线、作品分析
- **零依赖部署** 纯前端 + Node.js 内置模块，无需安装任何第三方包

> ✨ **所有处理在浏览器端完成，数据不上传服务器，完全离线可用。**

## 🎯 核心功能

### 1️⃣ 小说自动解析

| 特性 | 说明 |
|------|------|
| 📑 **章节识别** | 「第一章」「第1章」「Chapter 1」「序幕」「序幕」等常见格式 |
| 💬 **对话提取** | 自动识别「」『』"""""" 引号中的对话，关联说话人与上下文 |
| 🏠 **场景标记** | 「场景：」「地点：」「时间：」「天气：」结构化标记解析 |

### 2️⃣ 剧本智能转换

- **三幕剧结构映射** — 根据章节数量自动划分建置/对抗/解决三幕
- **多格式支持** — 电影剧本 / 电视剧本 / 舞台剧本 一键切换
- **角色系统** — 自动提取角色列表，按台词频次分配主角/配角定位

### 3️⃣ 可视化工作台

- **剧本预览** — 结构化展示幕、场景、对话与动作描写
- **角色管理** — 编辑角色名称、定位、性格特征与描述
- **场景时间线** — 按地点分组展示剧情发展脉络
- **作品分析** — 对话/描写比例、台词量排行、改编建议

### 4️⃣ 结构化导出

- **YAML 导出** — 标准剧本 YAML 格式，可直接编辑
- **JSON 导出** — 便于程序化处理
- **纯文本剧本** — 可读格式，适合分享与打印

## 📄 YAML Schema

转换后的剧本采用 **YAML 格式**，兼顾机器可读性与人工可编辑性。

`yaml
screenplay:
  metadata:
    title: "月下惊鸿"
    author: "佚名"
    date: "2026-06-08"
    format: "film"
    pageEstimate: 120

  characters:
    - id: "char_001"
      name: "沈孤鸿"
      role: "主角"
      totalLines: 42

  acts:
    - id: "act_1"
      title: "第一幕（建置）"
      scenes:
        - id: "scene_001"
          setting:
            location: "洛阳城外·听雨亭"
            time: "戌时"
          content:
            - type: "action"
              description: "夜色如墨，月华如水。"
            - type: "dialogue"
              speaker: "沈孤鸿"
              line: "阁下深夜相邀，所为何事？"
`

完整的 Schema 定义与设计原因详见 [schema/screenplay-schema.md](schema/screenplay-schema.md)。

## 🛠️ 技术栈

- **前端** — 纯原生 JavaScript + CSS3（零框架）
- **后端** — Node.js 内置 http / s / path 模块（零第三方依赖）
- **数据格式** — YAML（自定义轻量序列化器）
- **设计风格** — 深色主题、玻璃态设计、渐变与动效

### 设计决策

| 决策 | 原因 |
|------|------|
| **零 npm 依赖** | 克隆即用，无需安装，降低使用门槛 |
| **浏览器端处理** | 数据不上传，保护作者知识产权 |
| **纯原生前端** | 避免框架学习成本，保持极速加载 |
| **3 章以上自动三幕** | 经典剧本结构的自动化映射 |

## 🧪 使用建议

### 为获得最佳效果，在小说中添加标记

`markdown
场景：洛阳城外·听雨亭
时间：戌时
天气：月明星稀
`

### 剧本打磨流程

1. **V1 - 自动转换**：保留原文风格，获得剧本骨架
2. **V2 - 角色打磨**：完善角色性格与台词
3. **V3 - 结构优化**：调整场景顺序，优化节奏
4. **V4 - 定稿**：添加技术标注，准备拍摄
